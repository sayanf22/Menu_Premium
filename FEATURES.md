# AddMenu - Subscription Plans

## Advanced Plan (₹199/month)

Digital menu display for restaurants.

**Features:**
- Digital menu with categories
- Up to 50 menu items
- Toggle item availability
- QR code generation
- Basic analytics & menu views
- Advanced Bell Feature (Waiter/Water/Bill calls)
- Dark/Light mode
- Mobile responsive design

---

## Premium Plan (₹399/month)

Complete menu and ordering system.

**Features:**
- Everything in Advanced
- Unlimited menu items
- Real-time order management
- Order notifications with sound
- Customer feedback collection
- Order status tracking
- Advanced analytics
- Priority support

---

## Security

- Secure authentication (PKCE flow)
- Row Level Security (RLS) for data isolation
- Rate limiting & DDoS protection
- Input sanitization (XSS prevention)
- Razorpay secure payments
- HTTPS enforced
- CSP headers configured

---

## Responsive Design

Fully responsive across all devices:
- Desktop (1920px+)
- Laptop (1024px - 1919px)
- Tablet (768px - 1023px)
- Mobile (320px - 767px)

---

## Advanced Bell Feature

Available in both plans. Customers can request:
- **Waiter** - Call staff to table
- **Water** - Request water refill
- **Bill** - Request bill

Restaurant dashboard receives real-time notifications with sound alerts.
